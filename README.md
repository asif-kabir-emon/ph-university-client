## Run Code Loacally

1. **Clone the GitHub Repository on local machine:**
    
    ```sh
    git clone https://github.com/asif-kabir-emon/ph-university-client.git
    ```

2. **Install dependencies:**

    ```sh
    cd ph-university-client
    
    npm install
    ```

3. **Running the website:**
    ```sh
    npm run dev
    ```
